Mô hình hoạt động giữa client, Identity Provider và Service Provider

Mô hình giả ngữ

+--------+                      +-------+                       +--------+
| Client |                      |  IdP  |                       |   SP   |
|        |                      |       |                       |        |
+--------+                      +-------+                       +--------+
    |---------- get Token -------->>|                               |
    |                               |                               |
    |<<------- return Token --------|                               |
    |                               |                               |
    |                               |                               |
    |---- get SAML2 assertion ---->>|                               |
    |                               |                               |
    |<<-- return SAML2 assertion ---|                               |
    |                               |                               |
    |                               |                               |
    |-------------- submit SAML2 assertion ----------------------->>|
    |                               |                               |
    |---------------- obtain unscope token ----------------------->>|
    |                               |                               |
    |<<------------------ return token access ----------------------|
    |                               |                               |
    |                               |                               |
    |-------------- use Service via token access ----------------_>>|
    |                               |                               |
